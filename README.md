# discord_cot
Discord bot in C using the [concord](https://github.com/Cogmasters/concord) 
library

## building
```console
$ clang nob.c -o nob
$ ./nob build
```

alternatively use make

## running
```console
$ touch token
$ echo "YOUR_BOT_TOKEN" > token
$ ./nob run
```
alternatively for the last step, you could use
```console
$ build/nutte
```
